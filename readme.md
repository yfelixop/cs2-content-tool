# CS2 Content Tool

Simple tool to track ideas for CS2 highlight videos.

## Features
- Store content ideas
- View ideas in the terminal
- Simple Python script

## Usage

Run the tool:

python3 tool.py

Follow the menu:
1. Add idea
2. Show ideas
3. Exit

## Notes
- Ideas are not saved permanently yet
- This is a simple project for learning Git and basic tools
